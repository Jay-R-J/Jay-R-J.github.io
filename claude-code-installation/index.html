从零安装 Claude Code：国内通过 CC-Switch 接入 DeepSeek 完整教程

2026-06-24T02:00:00.000Z2026-06-24T02:00:00.000Z1.2k4 分钟China

引言：为什么需要这个教程？

Claude Code 是一款强大的终端 AI 编程助手，但由于网络限制，国内用户无法直接使用官方服务。本教程将教你如何通过 CC-Switch 工具，将 Claude Code 的请求转发到国内的 DeepSeek API，实现在国内顺畅使用 AI 编程辅助功能。

核心原理：Claude Code 和 DeepSeek 等模型使用的 API 格式不同，关键在于中间需要一个”翻译层”，把 Claude Code 发出的请求转换成 DeepSeek 能听懂的格式。CC-Switch 就是这个”翻译官”。

准备工作

安装 Node.js 18+

Claude Code 基于 Node.js 开发，需要 Node.js 版本 18 或更高。

  1. 访问 Node.js 官网
  2. 下载并安装 LTS 版本（推荐）或最新版本

提示：安装时请确保勾选”Add to PATH”选项，以便在命令行中可以直接使用 node 和 npm 命令。

安装 Git for Windows（Windows 用户）

Windows 用户需要安装 Git for Windows，以确保某些依赖可以正常安装。

  1. 访问 Git 官网
  2. 下载并安装最新版本
  3. 安装完成后验证：

配置 CC-Switch

第四步：下载 CC-Switch

  • 百度网盘下载 - 提取码: zxdr（点击链接后如未自动填充，请手动输入）

选择对应版本：

  • Windows 系统：下载 CC-Switch-v3.16.1-Windows.msi
  • macOS 系统：下载对应的 Mac 版本
  • Linux 系统：下载对应的 Linux 版本

本文以 Windows 系统为例进行演示。

第五步：安装 CC-Switch

  1. 双击下载好的 CC-Switch-v3.16.1-Windows.msi 文件
  2. 按照安装向导完成安装
  3. 安装完成后，点击启动 CC-Switch

第六步：添加 DeepSeek 模型

  1. 打开 CC-Switch 主界面
  2. 点击导航栏的 “Claude Code” 选项
  3. 点击右侧的 “+”（加号） 按钮
  4. 在模型列表中选择 “DeepSeek”
  5. 将复制好的 API key 粘贴到 “API key” 输入框
  6. 滚动到页面最底部
  7. 点击 “添加” 按钮

提示：如何获取 DeepSeek API Key？请参考我的上一篇文章《国内使用 Codex 完整教程：通过 CC-Switch 接入 DeepSeek》。

第七步：开启本地路由

这是最关键的一步！由于 Claude Code 和 DeepSeek 的 API 格式不同，必须通过路由进行转换。

  1. 返回 CC-Switch 主界面
  2. 点击左上角的 “设置” 图标（齿轮）
  3. 点击第二个标签页 “路由”
  4. 找到 “本地路由” 选项
  5. 打开路由总开关
  6. 在下拉菜单中选择 “Claude”
  7. 保存设置

原理解释：这一步启用了 API 格式转换功能，让 CC-Switch 能够将 Claude Code 的请求”翻译”成 DeepSeek 能理解的格式。

第八步：启动 DeepSeek 模型

  1. 回到 CC-Switch 主界面
  2. 找到刚才添加的 DeepSeek 模型
  3. 点击 “启动” 按钮
